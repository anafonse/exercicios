var nome = window.prompt('Qual seu nome?')
isso significa na prática que a variavel "nome" vai receber o resultado do prompt que aparecer na janela

JavaScript É CASE SENSITIVE, nunca esquecer dessa merda. Outra coisa pra sempre lembrar é que tudo que entre em window.prompt ("valor aqui inserido") é uma STRING


A receita pro desastre:

var num1 = window.prompt("Digite um numero")
var num2 = window.prompt("Digite outro numero")
var soma = num1 + num2

window.alert("O Resultado da soma dos dois numeros digitados é " + soma)


Esse é jeito ERRADO de fazer as somas. As variáveis num1 e num2 estão sendo lidas como STRING, e por isso não é realizada a soma entre elas.

O resultado disso é talvez un dos memes mais famosos da programação conteporanea 💀

E,bom, isso acontece simplesmente por que quando o window.alert vai mostrar o "resultado", ele exibe os valores digitados como strings (que é o que eles são, mesmo sendo um numero), e não como numeros por que não foi informado a ele que num1 e num2 seriam numeros e ora bolas como que o JavaScrip tvai adivinhar, então, por causa disso, o + ali esta servindo como o concatenador de duas stings em vez de um sinal de uma operação de adição. Rá. Simples.

// + é concatenação e adição
//(string + string) o + serve pra concatenação
//(number + number) o + vai servir para adição


Ta mas se não é assim, qual é o jeito correto?

Pra gente fazer do jeito certo a gente tem que saber que as variaveis precisam ser transformadas em numeros para que o window.prompt nao tranforme nossos numeros em strings!

var num1 = Number.parseInt(window.prompt("Digite um numero"))
var num2 = Number.parseInt(window.prompt("Digite outro numero"))
var soma = num1 + num2 // aqui, eles ja se tornaram numeros
window.alert("O Resultado da soma dos dois numeros digitados é " + soma)


Para que as variáveis fossem transformadas em numeros, usou-se o Number.parseInt antes de window.prompt que os transformou em numeros

O resultado:

ATENÇÃO !! 🚩

O Number.parseInt vai converter a variável pra um numero INTEIRO, se voce quiser um numero real (ou seja, com vírgula tipo 10,1) tem que usar o Number.parseFloat!

var num1 = Number.parseFloat(window.prompt("Digite um numero: "))
var num2 = Number.parseFloat(window.prompt("Digite um numero: "))
var soma = num1 + num2
window.alert("O Resultado da soma dos dois numeros digitados é " + soma)



OBSERVAÇÃO ~~EXTREMAMENTE~~ IMPORANTE!!!!!!!!
NÃO usar o "del" do teclado numerico, senão não funciona. Tem que usar o "." da tecla ">", ou "." do próprio teclado numerico mesmo DESDE QUE NAO SEJA O "." DO DA TECLA "DEL"!

Observação 2: se for necessario ter que converver de string pra numero, usa-se o String(var) ou só var.toString()
