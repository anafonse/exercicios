Aula sobre: Variáveis e títulos primitivos!
  • As variáveis têm tamanho diferentes, para caber dados de tamanhos diferentes;

  • A cadeia de caracteres (sefa cpf, rg, uma quencia de numeros comoNIS ou ate mesmo um nome) é chamada de STRING;

  • Craze, Apóstore e aspas têm difereça ao serem usadas ( ``, '', "")

NULL usado para indicar a ausencia de um objeto qualquer
UNDEFINED indica que nao tem/recebeu nenhum valor

Identificadores é como são chamados os nomes de cada variável e existem regras para nomea-los. São elas :
  • Podem começar com uma letra, $ ou _ ;

  • NÃO podem começar com numeros;

  • É POSSIVEL usar acentos e símbolos;

  • NÃO podem conter espaços, os espaços sao substtuidos por "_";

  • NÃO podem ser palavras RESERVADAS (funciton, alert, window, var), ou seja, nao pode usar palavras que o JavaScript usa como comando.
